URI formats
Local path file:///absolute/path/to/folder
AWS S3 s3://bucket-name/optional/prefix
Needs AWS_ACCESS_KEY_ID + AWS_SECRET_ACCESS_KEY (or instance profile).
Azure Blob az://container-name/optional/prefix
Needs AZURE_STORAGE_CONNECTION_STRING (or managed identity).
Google Cloud Storage gs://bucket-name/optional/prefix
Needs GOOGLE_APPLICATION_CREDENTIALS.
SharePoint sharepoint://site-name/Library/Folder
Requires the 'sharepoint' extra + SHAREPOINT_* env. Currently stubbed.

Heads-up on file:// — note three slashes: file:// + an absolute path that itself starts with /. The default file:///data is the Docker volume mount path; when running locally, point it at a writable folder on your disk.
